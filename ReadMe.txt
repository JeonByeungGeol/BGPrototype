========================================================================
    콘솔 응용 프로그램 : BGPrototype 프로젝트 개요
========================================================================

BGPrototype 프로그램에서 공통적으로 쓰일 수 있는 객체들을 만든 프로젝트 입니다.

공통적으로 쓰일 수 있는 객체들은 Log, Config, Session 등이 있습니다.

각각의 파일에 자세한 주석을 달았습니다.

추가로 필요한 부분이 있으면 지속적으로 업데이트 예정입니다.



/////////////////////////////////////////////////////////////////////////////
프로젝트 쓰임:

다른 프로젝트들에서 해당 객체들을 베이스로 사용할 수 있도록 만들었습니다.



/////////////////////////////////////////////////////////////////////////////
현재 추가한 객체:

BGConfigManager

BGLogManager

BGSessionManager
